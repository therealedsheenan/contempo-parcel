# contempo-parcel
Contempo using parcel

See demo here: [contempo-parcel.surge.sh](http://contempo-parcel.surge.sh)


## Installation
Clone the repository then:

```
$ npm i # yarn
```

Watch files

```
$ npm run watch i # yarn watch
```

Build for production

```
$ npm run bulld # yarn build
```
